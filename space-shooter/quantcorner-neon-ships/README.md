# QuantCorner Neon Ships

AI-generated neon sci-fi spaceship sprites for the QuantCorner dark terminal web UI.

Visual direction: dark graphite spacecraft, teal/mint neon edge lighting, HUD/terminal feel, matching the QuantCorner Tools page aesthetic.

## Contents

- `png-alpha/` — 10 transparent PNG sprites, 384x384. Best for flexible UI placement.
- `png-black/` — 10 PNG sprites on black background, 384x384. Best when preserving full glow against dark UI.
- `webp-alpha/` — transparent WebP versions.
- `webp-black/` — black-background WebP versions.
- `sheet-source.png` — original Image Generator sheet.
- `preview.png` — preview grid on dark background.
