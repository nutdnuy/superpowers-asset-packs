# QuantCorner Generated Ships

AI-generated 8-bit spaceship sprites for the QuantCorner Tools page.

- Source: generated with Image Generator for QuantCorner UI use.
- Style: cute 8-bit / arcade pixel art on transparent backgrounds.
- Contents: 10 PNG sprites, 10 WebP sprites, source sheet, transparent sheet, and preview.
- Intended use: `https://www.quant-corner.com/?tools=1` and related QuantCorner UI experiments.

## Raw URL pattern

```text
https://raw.githubusercontent.com/nutdnuy/superpowers-asset-packs/master/space-shooter/quantcorner-generated-ships/png/qc-8bit-ship-v2-01.png
```

Change `01` through `10` for each ship.
