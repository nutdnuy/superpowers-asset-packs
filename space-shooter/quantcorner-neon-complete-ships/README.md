# QuantCorner Neon Complete Ships

AI-generated complete neon sci-fi spaceship sprites for the QuantCorner Tools page.

This set replaces the earlier sheet-cropped neon ships. Each spacecraft was generated as a separate full-body image, then processed into centered UI-ready assets so wings, side pods, and engines are not clipped.

## Contents

- `png-alpha/` — 10 transparent PNG sprites, 512x512.
- `png-black/` — 10 black-background PNG sprites, 512x512.
- `webp-alpha/` — transparent WebP versions.
- `webp-black/` — black-background WebP versions.
- `source/` — original generated source images.
- `preview.png` — preview grid.

## Raw URL pattern

Transparent PNG:

```text
https://raw.githubusercontent.com/nutdnuy/superpowers-asset-packs/master/space-shooter/quantcorner-neon-complete-ships/png-alpha/qcn-complete-ship-01-alpha.png
```

Black-background PNG:

```text
https://raw.githubusercontent.com/nutdnuy/superpowers-asset-packs/master/space-shooter/quantcorner-neon-complete-ships/png-black/qcn-complete-ship-01-black.png
```

Change `01` through `10` for each ship.
